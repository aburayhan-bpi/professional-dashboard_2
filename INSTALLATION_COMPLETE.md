## 🎨 Customization

### Change Colors
Edit `src/app/globals.css` theme variables

### Add Pages
Create new folder in `src/app/dashboard/`

### Add Navigation
Edit `src/lib/constants.ts` SIDEBAR_ITEMS

### Connect API
Replace mock data with API calls
